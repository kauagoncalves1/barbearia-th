# Barbearia TH

Site profissional desenvolvido para a Barbearia TH, localizada em Campo Grande, Rio de Janeiro.

## Sobre o projeto

Site institucional completo com foco em conversao de clientes via WhatsApp. Desenvolvido com design dark luxury, responsivo para mobile e otimizado para SEO local.

## Paginas e secoes

- **Hero** — apresentacao impactante com chamada para agendamento
- **Quem Somos** — historia e diferenciais da barbearia
- **Servicos** — tabela de servicos e precos
- **Contato** — horarios de funcionamento, endereco com link para Google Maps e agendamento via WhatsApp

## Tecnologias utilizadas

- [Next.js 14](https://nextjs.org/) — framework React com App Router
- [Tailwind CSS](https://tailwindcss.com/) — estilizacao utilitaria
- [Vercel](https://vercel.com/) — hospedagem e deploy continuo
- [SupaBase](https://supabase.com/) — banco de dados
  
## Deploy

O site e hospedado na Vercel com deploy automatico a cada push na branch `main`.
